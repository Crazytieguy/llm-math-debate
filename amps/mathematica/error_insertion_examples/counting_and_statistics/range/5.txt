Given the following problem:

```

Find the range of the list:

$\left(-5,-3,-8,\frac{137}{16},\frac{6}{5},-4,3,-8,2,3,\frac{7}{8},0,5,3,\frac{28}{3}\right)$

```

And correct solution:

```

1. The range of a list is given by:

$(\text{largest element})-(\text{smallest element})$

2. $\left(-5,-3,-8,\frac{137}{16},\frac{6}{5},-4,3,-8,2,3,\frac{7}{8},0,5,3,\frac{28}{3}\right)\text{  We see that the largest element is }\frac{28}{3}$:

$\frac{28}{3}-(\text{smallest element})$

3. $\left(-5,-3,-8,\frac{137}{16},\frac{6}{5},-4,3,-8,2,3,\frac{7}{8},0,5,3,\frac{28}{3}\right)\text{  We see that the smallest element is }-8:$

$-(-8)+\frac{28}{3}$

4. $-(-8)+\frac{28}{3}=\frac{52}{3}$:

Answer: $\frac{52}{3}$

```

Insert an error into step 4.
