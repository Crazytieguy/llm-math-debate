Given the following problem:

```

Find the range of the list:

$\left(\frac{295}{32},7,\frac{97}{10},0,3,\frac{69}{8},\frac{111}{16}\right)$

```

And correct solution:

```

1. The range of a list is given by:

$(\text{largest element})-(\text{smallest element})$

2. $\left(\frac{295}{32},7,\frac{97}{10},0,3,\frac{69}{8},\frac{111}{16}\right)\text{  We see that the largest element is }\frac{97}{10}$:

$\frac{97}{10}-(\text{smallest element})$

3. $\left(\frac{295}{32},7,\frac{97}{10},0,3,\frac{69}{8},\frac{111}{16}\right)\text{  We see that the smallest element is }0:$

$-0+\frac{97}{10}$

4. $-0+\frac{97}{10}=\frac{97}{10}$:

Answer: $\frac{97}{10}$

```

Insert an error into step 3.
