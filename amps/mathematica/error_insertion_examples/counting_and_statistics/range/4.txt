Given the following problem:

```

Find the range of the list:

$\left(8,\frac{59}{7},\frac{4}{5},2,0,-\frac{20}{3},-\frac{23}{3},8\right) = \left(8,\frac{59}{7},\frac{4}{5},2,0,-\frac{20}{3},-\frac{23}{3},8\right)$

```

And correct solution:

```

1. The range of a list is given by:

$(\text{largest element})-(\text{smallest element})$

2. $\left(8,\frac{59}{7},\frac{4}{5},2,0,-\frac{20}{3},-\frac{23}{3},8\right)\text{  We see that the largest element is }\frac{59}{7}$:

$\frac{59}{7}-(\text{smallest element})$

3. $\left(8,\frac{59}{7},\frac{4}{5},2,0,-\frac{20}{3},-\frac{23}{3},8\right)\text{  We see that the smallest element is }-\frac{23}{3}$:

$-\frac{-23}{3}+\frac{59}{7}$

4. $-\frac{-23}{3}+\frac{59}{7}=\frac{338}{21}$:

Answer: $\frac{338}{21}$

```

Insert an error into step 2.
