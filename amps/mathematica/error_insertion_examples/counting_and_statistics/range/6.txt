Given the following problem:

```

Find the range of the list:

$\left(6,9,3,-6,9,-\frac{1}{7},-1,1,0,\frac{64}{7},1,-\frac{16}{3},5,-7\right) = \left(6,9,3,-6,9,-\frac{1}{7},-1,1,0,\frac{64}{7},1,-\frac{16}{3},5,-7\right)$

```

And correct solution:

```

1. The range of a list is given by:

$(\text{largest element})-(\text{smallest element})$

2. $\left(6,9,3,-6,9,-\frac{1}{7},-1,1,0,\frac{64}{7},1,-\frac{16}{3},5,-7\right)\text{  We see that the largest element is }\frac{64}{7}$:

$\frac{64}{7}-(\text{smallest element})$

3. $\left(6,9,3,-6,9,-\frac{1}{7},-1,1,0,\frac{64}{7},1,-\frac{16}{3},5,-7\right)\text{  We see that the smallest element is }-7:$

$-(-7)+\frac{64}{7}$

4. $-(-7)+\frac{64}{7}=\frac{113}{7}$:

Answer: $\frac{113}{7}$

```

Insert an error into step 1.
