Given the following problem:

```

Simplify the following:

$\sqrt{42}-26+\sqrt{71}-\sqrt{40}$

```

And correct solution:

```

1. $\sqrt{40} = \sqrt{2^3\times 5} = 2 \sqrt{2} \sqrt{5}$:

$\sqrt{42}-26+\sqrt{71}-2 \sqrt{2} \sqrt{5}$

2. $\sqrt{2} \sqrt{5}=\sqrt{2\times 5}$:

$\sqrt{42}-26+\sqrt{71}-2 \sqrt{2\times 5}$

3. $2\times 5 = 10:$

Answer: $\sqrt{42}-26+\sqrt{71}-2 \sqrt{10}$

```

Insert an error into step 3.
